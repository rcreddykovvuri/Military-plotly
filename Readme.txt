The datasets used in these dashboard are taken from 3 different sources and the data is cleaned using python.
nuclear.csv created using data from https://en.wikipedia.org/wiki/List_of_states_with_nuclear_weapons
trend.csv source: https://correlatesofwar.org/data-sets/national-material-capabilities
cleaned.csv source: https://www.kaggle.com/datasets/sleymanzeynul/military-strengths-of-countries-2021 

This is the information about the datasets after cleaning:

nuclear.csv
this contains information about the nuclear powers in the world:
country - country name
total - total nuclear bombs hold by the country
type - NPT (Non-Proliferation Treaty) or Non-NPT
iso - country code(iso3)

trend.csv
This contains data from 2005 to 2016:
country - country name
year - year of the data
military_exp - military expenditure in dollars
military_personnel - no of military expenditure of the country
total_population - total population 
iso - country code(iso3)
continent - continent of the country

cleaned.csv
This dataset consists of information of countries military power by 2021
country - country name
aircraft - number of aircrafts
fighter/interceptor - number of fighter/interceptor
attack_aircraft - number of attack_aircraft
transport_aircraft_fleet - number of transport_aircraft_fleet
trainer_aircraft_fleet - number of trainer_aircraft_fleet
helicopter_fleet - number of helicopter_fleet
attack_helicopter_fleet - number of attack_helicopter_fleet
tank - number of tanks
afv/apc - number of afv/apc
self-propelled_artillery - number of self-propelled_artillery
towed_artillery - number of towed_artillery
rocket_projector - number of rocket_projector
navy_fleets - number of navy_fleets
aircraft_carrier_fleet - number of aircraft_carrier_fleet
submarine_fleet - number of submarine_fleet
destroyer_fleet - number of destroyer_fleet
frigate_fleet - number of frigate_fleet
airport_totals - total airports in the country
available_manpower - availble man power in all the countries
total_population - total population 
iso - country code(iso3)
continent - continent of the country
